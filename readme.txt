Steps to run the program
1. Run ChatServer.java
2. Run ChatClient.java for as many clients as you want.
3. Enter a one word username for each client.
4. Type in a command as follows
 	  To broadcast a messagae : broadcast msg <text>
          To unicast a message: unicast msg <text> username
          To blockcast a message: blockcast msg <text> username
          To broadcast a file: broadcast file <filename>
          To unicast a file: unicast file <filename> username
          To leave: /quit
